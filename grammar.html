1. Phonology (Sounds)

The Draconic language features deep, resonant vowels and clipped consonants.

Vowels

A: [ɑ] as in "father"
E: [ɛ] as in "bed"
I: [i] as in "machine"
O: [ɔ] as in "stone"
U: [u] as in "moon"
Y: [ɪ] as in "rhythm"

Consonants

K: [k] - "kite"
G: [g] - "go"
S: [s] - "snake"
Z: [z] - "zebra"
R: [ʀ] - rolled R
TH: [θ] - "think"
SH: [ʃ] - "shush"
V: [v] - "vibe"
M: [m] - "moon"
L: [l] - "lunar"
N: [n] - "night"
CH: [ʧ] - "chime"
P: [p] - "pipe"
T: [t] - "tap"
D: [d] - "dog"

Diphthongs

AI: [aɪ] - "high"
EA: [ɛɪ] - "day"
OI: [ɔɪ] - "boy"
AU: [aʊ] - "out"
OU: [oʊ] - "how"

2. Morphology (Word Formation)

Draconic uses agglutination to build meaning from smaller morphemes.

Common Morphemes

Prefixes: 
Raak- = "Dragon"
Vura- = "Rune"
Urth- = "Ancient"
Dra- = "Power"
Azgar- = "Magic/Strength"
Suffixes: 
-haz = possessive
-voth = noun-forming
-rath = verb-forming
-kor = plural
Infixes: 
-th- = transformation/change

3. Syntax (Sentence Structure)

Word order is typically Subject-Object-Verb (SOV).

Raak’haz kalth’vak ir’raakh. — "The dragon breathes fire."
Vura’kalth vrakkar azgar. — "The rune binds power."

Verb Usage

Vura-rath = "to create a rune"
Vura’kalth rathor. = "The rune creates fire."

4. Sentence Construction Examples

Basic Sentences

Vorn’az thran ez? — "We see you?"
Vorn goth Karnaz. — "I’m going to the village."

Questions

Vek? — "What?"
Karn? — "Who?"
Goth? — "Where?"
Yrrn? — "When?"

5. Possessive and Plural Forms

Possession: 
Raak’haz = "Dragon’s"
Vura’kalth haz = "The Rune’s Flame"
Plurality: 
Raak-kor = "Dragons"
Zaroth’kor = "Dooms"

6. Tense and Aspect

Tense is often inferred from context or the use of -th- as an infix.

Vura-th-rath — "to create or transform a rune"
Vura’kalth thran — "The rune saw"

7. Word Building and Combining

Raak’thor — "Dragon’s rune"
Azgar’voth — "Power source"
Raak’gor — "Dragon’s strength"
Gor’velth — "Big strength"

8. Draconic Sentence Structure Practice

Ez goth Zal’nirrn. — "You go evening."
Vorn velth, zorak. — "I’m strong, friend."

9. Crash Course: Speaking the Draconic Tongue

Pronunciation Rules

A = ah, E = eh, I = ee, O = oh, U = oo
R = rolled, ' = glottal stop, double consonants = emphasized

Sentence Structure

Subject – Verb – Object (SVO)

Vorn zar veth. — "I am good."
Ez vek marnak. — "You want food."

Core Patterns

[Noun]’[Modifier] — Raak’haz = "Dragon’s"
[Verb][Object] — Thran marnak = "Look at the food"

Practice Phrases

Veth’ra — "Hello"
Vek’ra thrak? — "What is that?"
Ez velth? — "Are you strong?"
Vorn vek raruoki. — "I want music."
Zaroth’ra — "Goodbye"
