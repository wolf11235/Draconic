2. Morphology (Word Formation)

Draconic is agglutinative, meaning words are built by combining smaller parts (morphemes) to express complex ideas.

Common Morphological Elements

Prefixes: (meaning, ownership, power)

Raak- = Dragon
Vura- = Rune
Urth- = Ancient
Dra- = Power
Azgar- = Magic or Strength

Suffixes: (function, tense, possession, plurality)

-haz = Possessive → Raak’haz = Dragon’s
-voth = Noun-forming → Azgar’voth = Power source
-rath = Verb-forming → Rathor = To create
-kor = Plural → Raak’kor = Dragons

Infixes: (modification)

-th- = Change, past tense, transformation
→ Vura-th-rath = To transform a rune

3. Syntax (Sentence Structure)

Draconic follows Subject–Object–Verb (SOV) order.

Raak’haz kalth’vak ir’raakh. → The dragon breathes fire.
Vura’kalth vrakkar azgar. → The rune binds power.

4. Sentence Construction Examples

Basic

Vorn’az thran ez? → Are we seeing you?
Vorn goth Karnaz. → I'm going to the village.

Questions

Vek? = What?
Karn? = Who?
Goth? = Where?
Yrrn? = When?
Thran? = How?

5. Possession & Plurality

Possessive: -haz → Raak’haz = Dragon’s
            Vura’kalth’haz = The rune’s flame

Plural: -kor → Raak’kor = Dragons
           Zorak’kor = Friends

6. Tense & Aspect

Draconic has minimal verb conjugation. Time is often implied through context or infix use.

-th- infix indicates past or change:

Thran = Sees
Th-thran = Saw

Example:
Vura’kalth thran. → The rune saw / was seen.

7. Word Building (Compounds & Combinations)

Raak’thor = Dragon’s rune
Azgar’voth = Power source
Raak’gor = Dragon’s strength
Gor’velth = Great strength

8. Practice Sentences (with Gloss)

Ez goth Zal’nirrn. → You go evening. (You are going in the evening.)
Vorn velth, zorak. → I’m strong, friend.
